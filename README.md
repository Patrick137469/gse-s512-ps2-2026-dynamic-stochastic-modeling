# GSE-S512 Problem Set 2 (2026)

Public submission repository for **GSE-S512: Dynamic Stochastic Modeling**, focused on:
- weekly economic index analysis,
- time-series stationarity and AR modeling,
- state discretization,
- Markov chain estimation and ergodic distribution analysis.

## Repository Structure

- `Problem_Set_2_2026.ipynb` - main assignment notebook with code, outputs, and figures.
- `requirements.txt` - Python dependencies used by the notebook.

## Assignment Coverage

The notebook includes:
- Problem 1: Weekly Economic Index
- Problem 2: Stationarity and linear time-series analysis
- Problem 3: Distributional properties
- Problem 4: Discretization into states
- Problem 5: State-dependent means
- Problem 6: Markov transition matrix estimation
- Problem 7: Markov chain properties
- Problem 8: Ergodic (stationary) distribution
